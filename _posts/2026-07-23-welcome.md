This is the first note on the site. I will use this space to share project updates, ideas, and writing in progress.

To publish a post, create a Markdown file in `_posts` named:

`YYYY-MM-DD-your-title.md`

Give it a title in the front matter, then write the post beneath it. The site will handle the URL, date, and layout automatically.
